## 快速链接

### 性能基准

```bash
# 运行所有基准测试
make benchmark

# 运行特定基准测试
go test -bench=BenchmarkAgentCreation -benchmem ./pkg/agno/agent/

# 生成 CPU profile
go test -bench=. -cpuprofile=cpu.out ./pkg/agno/agent/
```

## 设计原则

### KISS (Keep It Simple, Stupid)

Agno-Go 拥抱简单性:

- **专注范围**: 3 个 LLM 提供商 (OpenAI, Anthropic, Ollama) 而不是 8+
- **核心工具**: 5 个核心工具而不是 15+
- **清晰抽象**: Agent, Team, Workflow
- **最小依赖**: 优先使用标准库

### 性能优先

Go 的并发模型使得:

- 原生 goroutine 支持并行执行
- 无 GIL (全局解释器锁) 限制
- 高效的内存管理
- 编译时优化

### 生产就绪

为实际部署而构建:

- 全面的错误处理
- 上下文感知的取消
- 结构化日志
- OpenTelemetry 集成
- 健康检查和指标
